# Panduan Pengguna KLSI 4.0

Folder ini menyimpan materi onboarding untuk mahasiswa, dosen, dan peneliti.

## Struktur File

- `student_profile.md` – versi default (Bahasa Indonesia) panduan membaca profil.
- `educator_responsible_use.md` – versi default panduan penggunaan bertanggung jawab.
- Jika tersedia terjemahan, gunakan sufiks kode bahasa ISO-639-1:
  - `student_profile.en.md`
  - `educator_responsible_use.en.md`

## Workflow Lokalisasi

1. Tulis atau perbarui dokumen default (ID).
2. Ajukan terjemahan via PR terpisah; sertakan reviewer bilingual dan scientific lead.
3. Perbarui tabel di bawah ketika terjemahan selesai.

| Dokumen | Bahasa Tersedia | Catatan |
|---------|-----------------|---------|
| student_profile | ID, EN | Default in Indonesian; English translation available. |
| educator_responsible_use | ID, EN | Default in Indonesian; English translation available. |

Tambahkan baris baru ketika terjemahan lain (EN, etc.) siap.
